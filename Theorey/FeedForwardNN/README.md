# Deep Learning Specialization Notes (Andrew Ng)

This repository contains my summarized notes and presentation slides from the first two courses of the **Deep Learning Specialization** by Andrew Ng.

The purpose of this project is to consolidate key concepts from the course into a structured and easy-to-review format. The material is organized as a presentation to serve as a quick reference for important ideas in deep learning and neural network fundamentals.

## Courses Covered

1. Neural Networks and Deep Learning  
2. Improving Deep Neural Networks: Hyperparameter Tuning, Regularization and Optimization  

## Contents

The presentation includes summarized explanations of:

- Fundamentals of neural networks
- Forward propagation and backpropagation
- Activation functions
- Loss functions
- Gradient descent and optimization algorithms
- Bias–variance tradeoff
- Regularization techniques
- Hyperparameter tuning strategies
- Practical tips for training deep neural networks

The slides are intended to capture the **core intuition and mathematical understanding** behind these concepts rather than reproducing the full course material.

## Purpose of This Repository

This repository is part of my learning portfolio in **Artificial Intelligence and Scientific Machine Learning**. It documents my learning process and serves as a reference for future projects involving deep learning and physics-based machine learning applications.
